# daily-ai-brief(自动生成区 · 请勿手动整理)

本目录由自动化流水线(Horizon 三层 AI 解读)每日生成,**不属于手动维护的笔记体系**。

- 维护知识库时(包括 Claude Code、孤儿笔记检查、链接整理等流程),**请整体跳过此目录**。
- 本区文件不应被纳入图谱双链、概念抽取、MOC 索引等手动笔记机制。
- daily 子目录为易腐区,仅保留最近约 79 天,更旧的每日简报会被自动清理。
- 真正值得长期留存的是 monthly 子目录(月度综述)。

目录说明:

- index.md — 每日简报索引,最新在最上
- daily — 易腐区:每日简报,按年/月归档,自动清理(约79天)
- _synthesis — 第三层整体分析(月度综述的原料)
- monthly — 保鲜区:月度综述,长期沉淀
